角色：你是一个精通 Vue 3 (使用 Composition API) 和 Tailwind CSS 的资深前端开发专家。

任务：请帮我编写一个位于网页“右侧面板（Right Sidebar Panel）”的音乐详情组件。该组件整体为深色主题（Dark Theme），请严格按照以下结构、样式和逻辑进行编码，并确保组件具有良好的复用性（通过 props 传入数据）。

---

### 一、 整体布局与样式
- 位置：固定在网页右侧面板（Width 约为 300px - 350px），垂直排列（Flex-col），带有内边距（Padding）。
- 背景：深色背景，文字为白色或浅灰色。

### 二、 组件内容结构（由上至下）

1. 【面板顶部标题】
   - 显示顶部第一艺术家的名字，字体小巧、加粗，带下划线样式。

2. 【歌曲封面】
   - 紧跟在标题下方，正方形大图，四周有微小的圆角（Rounded）。

3. 【歌曲主信息栏】
   - 采用 Flex 布局，两端对齐（justify-between），垂直居中。
   - 左侧包含：
     - 歌曲名字：字体较大、加粗、白色。
     - 歌曲创作者名字：字体较小、灰色，支持显示联名创作者（如：mimichi, datealyfe）。
   - 右侧包含【添加到收藏歌单按钮】：
     - 需要根据状态（isFavorite: boolean）进行条件渲染（v-if / v-else）：
       - 如果【已收藏】（isFavorite === true）：显示一个圆形、绿色底、中间带黑色对勾 "√" 的图标。
       - 如果【未收藏】（isFavorite === false）：显示一个由以下两个 SVG 路径组合而成的图标：
         - 路径 1: "M11.999 3a9 9 0 1 0 0 18 9 9 0 0 0 0-18m-11 9c0-6.075 4.925-11 11-11s11 4.925 11 11-4.925 11-11 11-11-4.925-11-11"
         - 路径 2: "M17.999 12a1 1 0 0 1-1 1h-4v4a1 1 0 1 1-2 0v-4h-4a1 1 0 1 1 0-2h4V7a1 1 0 1 1 2 0v4h4a1 1 0 0 1 1 1"

4. 【“关于艺人”卡片模块】
   - 整体外观：这是一个通用的卡片组件，外部由一个【深灰色、圆角矩形（Rounded-lg）】的盒子包裹，内部有适当的 Padding。
   - 卡片内部结构：
     - 顶部文本：固定显示“关于艺人”四个字，字体较小，灰色或半透明白。
     - 艺人头像：圆形图片（v-avatar），显示第一艺术家的头像。
     - 艺人名字：紧跟在头像下方，字体加粗，带下划线样式，显示第一艺术家的名字。
     - 听众人数：显示文本如“每月有 7,991 名听众”（具体数字需从 props 动态获取）。
     - 【关注按钮】：位于卡片的右下角或合适位置。目前仅作为界面摆设，不需要实现点击逻辑。样式为圆角胶囊状（Rounded-full），边框或背景明显。

### 三、 技术要求
1. 数据（如歌曲名、创作者、封面图片、每月听众数、是否收藏等）必须定义为 `props` 或可响应式的 `ref`，以便复用已有功能。
